Concept

Design an intelligent application:

  • that supports employees in their daily work and
  • allow CMC to better understand the needs of employees
  • so that employees have a positive experience with CMC.

Features

  • Welcome introduction for new employee
  • Onboarding + offboarding checklist
  • Upcoming events calendar + notification
  • Document templates
  • Submit timesheet
  • Contact directory
  • News feed
  • Mood tracker
  • Many more...

Platform

Mobile Browsers
Android Chrome
iOS Safari

Princples

Lean Startup Methodology

The practice of developing products and businesses based on 'validated learning', getting customer feedback early and often.

Build > Measure > Learn

User-centred design

Focus on the needs and wants of the users to create an application that is easy to use.

  1. Explicit understanding of users, tasks and environment.
  2. Involve users throughout the design and development.
  3. The design addresses the whole user experience.

Approach

Initiate

Start internally, with CMC consultants as our users.

  1. Form user group
  2. Define list of features
  3. Select 2 features > Develop > Design > Test

Review

Then, consider whether the application has enough features to be released internally and the market.

  1. Product viability
  2. Usability
  3. Market demand
